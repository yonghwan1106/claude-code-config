# 기사 유형별 구조 가이드

## 1. 스트레이트 뉴스 (Straight News)

핵심 사실을 신속·정확하게 전달하는 기본 기사.

### 구조: 역피라미드

```
[제목] - 핵심 내용을 15자 내외로 압축
[부제] - 제목을 보완하는 추가 정보 (선택)
[리드] - 육하원칙(5W1H) 중 핵심 2~3개를 1~2문장에 집약
[본문]
  - 1단락: 리드 보충 설명
  - 2단락: 배경/맥락
  - 3단락: 관계자 발언 (직접인용)
  - 4~n단락: 부가 정보 (중요도 내림차순)
[꼬리] - 향후 전망 또는 후속 보도 예고 (선택)
```

### 문체 특성
- 과거형 중심 (`~했다`, `~밝혔다`, `~전했다`)
- 객관적·건조한 톤
- 한 문장 40자 이내 권장
- 기자 의견 배제

### 리드 작성 공식
- **Who+What**: "OOO 시장이 ~을 발표했다"
- **What+Why**: "~이 시행된다. ~때문이다"
- **When+What**: "오는 O일부터 ~이 달라진다"

## 2. 기획 기사 (Feature/In-depth)

특정 이슈를 심층 분석하는 기사.

### 구조: 주제 전개형

```
[제목] - 분석적 시각을 담은 제목
[부제] - 핵심 분석 포인트
[리드] - 문제 제기 또는 핵심 발견
[본문]
  - 현황/문제점 제시
  - 원인 분석
  - 사례/데이터 근거
  - 전문가 의견 (2인 이상 권장)
  - 해외 사례 또는 비교 (선택)
  - 대안/전망
[맺음] - 정리 및 시사점
```

### 문체 특성
- 현재형+과거형 혼합
- 분석적 톤 (단, 주관 배제)
- 데이터/수치 활용 필수
- 단락 간 논리적 연결 중시

## 3. 인터뷰 기사

인물의 발언을 중심으로 구성하는 기사.

### 구조

**A. 일문일답형 (Q&A)**
```
[제목] - 인터뷰이 핵심 발언 인용
[리드] - 인터뷰이 소개 + 인터뷰 배경
[인터뷰이 프로필 박스]
[Q&A 본문]
  - Q: 질문 (간결하게)
  - A: 답변 (핵심 부분 굵게 처리)
```

**B. 서술형**
```
[제목]
[리드] - 인상적인 발언 또는 장면 묘사
[본문]
  - 만남 장면/분위기
  - 주제별 발언 정리 (직접인용 + 간접인용 혼합)
  - 기자 관찰/해석 (최소한)
[맺음] - 마지막 인상 또는 핵심 메시지
```

### 인용 규칙
- 직접인용: 큰따옴표 `"~"` 사용, 원문 그대로
- 간접인용: `~라고 말했다`, `~고 설명했다`
- 핵심 발언은 반드시 직접인용

## 4. 칼럼/사설 (Opinion/Editorial)

기자 또는 외부 필진의 의견을 담는 글.

### 구조

```
[제목] - 주장이나 관점을 담은 제목
[필자 소개] - 이름, 소속, 직함
[도입] - 화두 던지기 (일화, 통계, 질문 등)
[본문]
  - 현상/문제 인식
  - 논거 1 + 근거
  - 논거 2 + 근거
  - 반론 검토 (선택)
[결론] - 주장 정리 및 제언
```

### 문체 특성
- 1인칭 허용 (`필자는`, `~라고 본다`)
- 주장+근거 병행
- 설득적이되 감정 과잉 지양
- 명확한 결론 필수

## 5. 르포/현장 기사 (On-the-spot Report)

현장을 직접 방문하여 생생하게 전하는 기사.

### 구조

```
[제목] - 현장감을 살린 제목
[리드] - 현장 장면 묘사로 시작
[본문]
  - 시간순 또는 공간순 전개
  - 오감 묘사 (시각, 청각, 후각 등)
  - 현장 인물 발언
  - 기자 관찰 (주관 최소화)
  - 배경 설명 (현장 서술 사이에 삽입)
[맺음] - 현장 인상 또는 의미 부여
```

### 문체 특성
- 현재형 중심 (`~하고 있다`, `~이 보인다`)
- 묘사적·감각적 문체
- 짧은 문장으로 속도감
- 현장음/대화 직접인용 활용

---

## 6. 사건/사고 기사

사건·사고 발생 사실을 전하는 기사.

### 구조: 역피라미드 (엄격)

```
[제목] - 피해 규모/핵심 사실
[리드] - 언제, 어디서, 무슨 일이, 피해 규모
[본문]
  - 사건 경위 (시간순)
  - 피해 상황
  - 수사/대응 현황
  - 관계기관 입장
  - 목격자/관계자 발언
[후속] - 수사 방향, 향후 전망
```

### 주의사항
- 추측성 표현 금지
- 피해자 신원 보호 (2차 피해 방지)
- 경찰/소방 등 공식 발표 기반
- `~로 알려졌다`, `~인 것으로 전해졌다` 등 확인되지 않은 사실 구분
