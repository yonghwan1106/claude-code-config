# 인터넷언론 SEO 최적화 가이드

## 제목 최적화

### 원칙
- **길이**: 20~35자 (네이버 기준 모바일 노출 최적)
- **핵심 키워드를 앞쪽에 배치** — 검색엔진이 제목 앞부분에 가중치 부여
- **숫자 활용** — `3가지 이유`, `50% 증가` 등 구체적 수치가 CTR 높임
- **지역명 포함** — 로컬 뉴스 특성상 `용인`, `수원`, `경기도` 등 지역명이 검색 유입 핵심

### 제목 패턴

| 패턴 | 예시 |
|------|------|
| 지역+핵심사실 | `용인시, 올해 복지예산 20% 확대` |
| 인물+행위 | `김OO 시장 "신도시 교통 대책 마련"` |
| 수치+변화 | `경기도 출생률 역대 최저...전년比 15%↓` |
| 질문형 | `수원 광교 집값, 왜 다시 오르나` |

### 피해야 할 제목
- 낚시성/과장: `충격!`, `경악!`, `알고보니...`
- 모호한 표현: `이것이 문제다`, `결국 이렇게 됐다`
- 지나친 길이: 40자 초과

## 본문 SEO

### 키워드 배치
- **리드(1단락)**: 주요 키워드 1~2회 자연스럽게 포함
- **소제목**: 키워드 변형 활용 (동의어, 관련어)
- **본문 중반**: 키워드 밀도 1~2% 유지 (과도한 반복 금지)
- **마지막 단락**: 핵심 키워드 1회 재언급

### 구조 최적화
- **소제목 활용**: 500자마다 소제목으로 구분 → 가독성 + 검색 노출
- **볼드 처리**: 핵심 키워드/문장에 **굵게** 처리
- **리스트**: 나열형 정보는 불릿/번호 리스트로 정리
- **링크**: 관련 기사 내부 링크 2~3개 제안

### 본문 길이
| 기사 유형 | 권장 길이 |
|-----------|-----------|
| 스트레이트 | 800~1,500자 |
| 기획 | 2,000~4,000자 |
| 인터뷰 | 2,000~3,500자 |
| 칼럼 | 1,200~2,000자 |
| 르포 | 1,500~3,000자 |

## 메타 정보

기사 작성 시 함께 제안할 항목:
- **메타 디스크립션** (검색 결과 미리보기): 80~120자, 핵심 내용 요약
- **태그/키워드**: 5~8개, 주제·지역·인물 키워드 혼합
- **카테고리 제안**: 정치, 경제, 사회, 문화, 생활, 오피니언 등

## 사진 캡션 작성법

### 원칙
- 사진 속 상황을 **구체적으로** 설명 (누가, 무엇을, 어디서)
- 촬영 일시/장소 포함
- 사진 제공처 명시 (필요 시)
- 15~40자 권장

### 캡션 형식
```
[사진 설명]. 촬영일/장소. ⓒ제공처
```

### 예시
- `용인시청 대회의실에서 열린 2026년 예산심의회 현장. 2026.2.5. ⓒ경인블루저널`
- `김OO 시장이 주민과 대화하고 있다. ⓒ용인시 제공`
- `공사가 진행 중인 수원역 환승센터 전경. ⓒ경인블루저널`
